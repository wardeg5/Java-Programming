Due: Fri July 17 by 11:59pm

Summary:

  1. Solve some CodingBat problems.
  2. Update the clock display project to support a 12-hour clock and an alarm.

  1. Go to CodingBat.com/Java and to the Logic-1 set of problems.
  2. Complete any 5 of the problems in Logic-1 (4 pts each). The ones with 'H' next to them have a hint, so you may want to start with one of those.
    • When you complete a problem, the check mark turns green.
    • I recommend you create a free account so you can save your progress.
  3. Take a screen shot showing at least 5 green check marks and save it as LastName-a5-cb.png (or any common format such as pdf, jpg, etc.) using your last name.

[80 pts] Part 2: Alarm Clock

  1. Open the clock-display project in BlueJ and save it as LastName-a5-clock, e.g., Smith-a5-clock.
  2. Modify the ClockDisplay class to support a 12-hour clock as described in Exercise 3.31, including 'am' or 'pm' in the display. You will need to think about how to do this. There is more than one way, and you may choose any way you like, as long as you modify only the ClockDisplay class. DO NOT modify the NumberDisplay class. Read Exercise 3.32 to get some ideas.
    • If you need help, watch the textbook video note on solving the 12-hour clock exercise. However, you should spend significant time on this yourself before watching the video - this is a tricky exercise and your problem-solving skills will be improved by giving it your full effort before watching the video.
  3. Now add an alarm clock feature to the clock (once again, update only the ClockDisplay class). You need to decide what additional fields are required. The clock should have the following public methods (names & parameters exactly as shown) and functionality:
    • turnAlarmOn(int hour, int minute) - turns the alarm on and sets it to the given time
    • turnAlarmOff() - turns the alarm off
    • When the clock is created, the alarm should start in the 'off' position.
    • When the clock ticks to the set alarm time, if the alarm is turned on, an alarm should ring. You can simulate the ring with a print message.
    • If you have questions about how the alarm should function, post in the Week 4 discussion topic and the instructor will answer. It is not uncommon in software development that a specification is unclear; asking clarifying questions is an important skill to develop.
  4. As usual, update the header comment block and document your changes as you work.
  5. As usual, adhere to Java style guidelines as described in Appendix J.
  6. Test your code thoroughly! Make sure that each tick of the clock works as expected. This can be tricky, so spend some time thinking about test cases that cover all the relevant scenarios. Use the Inspector in BlueJ to examine the object's fields as you test.
  7. When you have completed the assignment, create a jar file of your project.
    1. While the project is open in BlueJ, choose Project->Create Jar File...
    2. Check the "Include source" check box
    3. Name the file LastName-a5-clock using your last name
