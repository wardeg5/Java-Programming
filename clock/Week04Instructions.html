Activities

  1. Read Chapter 3 while sitting in front of your computer and work through the exercises as you read.
  2. Read Appendix B: Java data types.

  4. Complete the following security module from Security Injections @ Towson: Integer Error. The first time you use the site, you will need to register. There is nothing to turn in, but this content is testable on the midterm and final exams.

    • The exam opens Wednesday 7/15 and must be completed by 11:59pm Wednesday 7/22.

Learning Objectives

  • Describe the concepts of abstraction and modularization and their advantages for object-oriented programming.
  • Distinguish between primitive types and object types in Java.
  • Identify code that could be vulnerable to integer error.
  • Draw an object diagram showing variables and objects in memory at run time.
  • Write code that creates and uses objects.
  • Use the special value 'null' appropriately in a program.
  • Use logical operators to build complex logical expressions.
