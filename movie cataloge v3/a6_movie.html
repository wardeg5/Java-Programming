[80 pts] Part 2: Movie Library

  1. Download the A6 movie project jar file from Content->Week 6->LastName-a6-movie.jar. (This link will not work for you until you have received feedback on A4. Email the instructor if you think you should be able to access it but can't.) Rename the file substituting your last name, e.g., Smith-a6-movie.jar. Open the project in BlueJ.
  2. Create a new class in the project and name it MovieLibrary. (Use the "New Class..." button.)
    1. Add a private field called movies whose type is ArrayList<Movie>. (Don't forget the import statement at the top so that you can use the ArrayList class.)
    2. Write a constructor that takes no parameters and creates a new ArrayList<Movie> assigning it to the movies field.
    3. Create an addMovie method that takes a title and run length as parameters. The method should create a new Movie object using the given parameters and add that Movie object to the movies list.
    4. Create a parameterless printMovies method. It should print information on all the movies in the library, according to the following specifications:
      • Print a row of symbols (e.g., "#################") as the first and last statement of the method to make clear where the output starts and ends.
      • If there are no movies in the library, print a message stating such (e.g., "No movies in the library")
      • Otherwise, loop over every movie and print the information on that movie. Remember that the Movie class has a printMovieInfo method! Call this method rather than duplicating the code to print the movie's attributes. Here's a loop you can use:
        for (Movie movie : movies) {
                   put 1 statement here that calls printMovieInfo on the movie object
        }
      • Here is a hint on how to structure the printMovies method. Only look at the hint after you've given it your full effort. Learning takes place as you struggle with the material, even when you don't solve it on your own.
  3. Test your changes as you go to make sure everything works.
  4. Document and use good style (Appendix J) as usual.
  5. When you have completed the assignment, create a jar file of your project.
    1. While the project is open in BlueJ, choose Project->Create Jar File...
    2. Check the "Include source" check box
